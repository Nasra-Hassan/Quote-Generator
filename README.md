# Quote Generator

A simple and elegant web application that generates random quotes from an API and allows users to share them on Twitter (X). Built using HTML, CSS, and JavaScript.


## Live Features

- Displays a random quote on page load
- Fetches quotes from an external API
- Generates a new random quote on button click
- Shows author name (or "Unknown" if missing)
- Tweet/share quote via Twitter (X) button
- Clean and responsive UI design


## Built With

- HTML5
- CSS
- JavaScript
- Fetch API
- Google Fonts
- DummyJSON Quotes API


## API Used

Quotes are fetched from: "https://dummyjson.com/quotes"
